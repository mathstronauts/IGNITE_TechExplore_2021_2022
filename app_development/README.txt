README

The Mars Earth Weather App is the Ignite Program Intro Project.

Students will create an app using Python.

Graphics will be displayed using the Pygame library and data will be extracted from the OpenWeatherMap API and data from REMS on the Mars Curiosity Rover.

The app will display the following data:

Main Menu
- app logo
- Earth
- Mars

Earth Page
- location
- date
- last updated time
- current temperature (Celsius)
- high/low temperature (Celsius)
- weather forecast (Clear, Rain, Snow, Clouds)
- predicted cloud type (based on a cloud base height function - cirro, strato, cumulo-form)
- show more button to display: sunrise, sunset, air pressure (mb), UV index, humidity (%), wind speed (km/h) 

Mars Page
- location
- sol (Mars Date)
- terrestrial date
- last updated time
- air temperature high/low (Celsius)
- ground temperature high/low (Celsius)
- weather forecast (sunny, cloudy, windy)
- show more button to display: sunrise, sunset, air pressure (mb), UV index
